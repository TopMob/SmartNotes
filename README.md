# SmartNotes

SmartNotes — современное веб-приложение для создания и управления заметками с поддержкой PWA, облачного хранения и расширенного редактора.

Проект спроектирован как офлайн-дружелюбное приложение с модульной архитектурой, готовое к установке на Desktop и Mobile.

## Основные возможности

- Создание, редактирование и удаление заметок
- Расширенный редактор с форматированием, медиа и задачами
- Теги, папки, закрепление и архив
- Защита заметок паролем
- Поиск по содержимому
- Темы оформления
- Облачное сохранение через Firebase
- Экспорт и импорт заметок
- Рисование и редактирование изображений
- Установка как PWA приложение
